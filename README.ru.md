# TestID Highlighter

Браузерное расширение для Chrome (Manifest V3), которое подсвечивает элементы на странице с атрибутами автотестов.

## Возможности

- **Подсветка элементов** с популярными атрибутами автотестов:
  - `data-test-id`, `data-cy`, `data-qa`, `data-test`, `data-testid`, `data-hook`, `data-e2e`

- **Умная подсветка:**
  - Минимальная подсветка по умолчанию (тонкий контур)
  - Усиленная подсветка при наведении
  - Тултип с атрибутом и подсказкой для копирования (использует Popover API - всегда поверх)

- **URL-активация:** Добавьте страницы в список разрешённых - подсветка работает автоматически на этих доменах

- **Поддержка wildcard:** Используйте `*.example.com` для поддоменов

- **Управление атрибутами:**
  - Добавление/удаление атрибутов
  - Drag & drop для изменения приоритета
  - Включение/выключение атрибутов без удаления (через чекбокс)

- **Копирование в буфер обмена:** нажмите Ctrl+C при наведении на подсвеченный элемент

- **Плавающая кнопка-глаз:**
  - Появляется автоматически на разрешённых страницах
  - Клик - включить/выключить подсветку
  - Перетаскивание для изменения позиции (позиция сохраняется)
  - Показывает текущее состояние (👁️ включено / 👁️‍🗨️ выключено)

- **Динамическая подсветка:** автоматически обновляется при изменениях на странице (AJAX, динамический контент)

- **SPA поддержка:** работает с React, Vue, Angular и другими SPA фреймворками

- **Многоязычность:** поддержка английского, русского и китайского языков с автоопределением языка браузера

- **Синхронизация:** настройки синхронизируются между устройствами через Chrome аккаунт

## Установка

1. Откройте `chrome://extensions/` в браузере Chrome
2. Включите "Режим разработчика" (переключатель в правом верхнем углу)
3. Нажмите "Загрузить распакованное расширение"
4. Выберите папку `testid-highlighter`

## Использование

1. Нажмите на иконку расширения в панели инструментов браузера
2. Добавьте текущую страницу в список разрешённых (кнопка "Добавить текущую") или введите URL вручную (поддерживаются wildcards вида `*.example.com`)
3. Элементы с атрибутами тестов на разрешённых страницах будут подсвечиваться автоматически
4. (Опционально) Настройте список атрибутов - добавляйте, удаляйте, меняйте порядок, включайте/выключайте

### Копирование атрибута

1. Наведите курсор на подсвеченный элемент
2. Появится тултип с атрибутом (например: `data-test-id="loginButton"`)
3. Нажмите `Ctrl+C` для копирования в буфер обмена

### Управление URL

- **Добавить текущую:** Добавляет текущий домен в список разрешённых
- **Добавить вручную:** Введите URL вручную (поддерживаются wildcards типа `*.example.com`)
- **Редактирование:** Кликните по URL в списке для редактирования
- **Удаление:** Кликните ✕ для удаления из списка
- **Автоскрытие:** Кнопка "Добавить текущую" скрывается, когда текущая страница уже в списке

### Управление атрибутами

- **Добавить:** Введите название атрибута и нажмите +
- **Удалить:** Кликните ✕ для удаления
- **Переключить:** Используйте чекбокс для включения/выключения атрибута без удаления
- **Сортировка:** Drag & drop для изменения приоритета (вверху = высший приоритет)

### Плавающая кнопка-глаз

На разрешённых страницах в правом нижнем углу появляется плавающая кнопка:
- **Клик** - включить/выключить подсветку
- **Drag** - перетащить кнопку в любое место экрана
- Позиция сохраняется и восстанавливается после перезагрузки страницы

### Смена языка

1. Выберите язык из выпадающего списка в правом верхнем углу popup
2. Интерфейс и подсказки тултипа обновятся на выбранном языке

## Как это работает

### Content Script (content.js)
- Автоматически загружается на все страницы при `document_idle`
- Читает настройки из `chrome.storage.local`
- Проверяет, находится ли текущий домен в списке разрешённых URL
- Применяет CSS класс `.testid-highlighter` для подсветки
- Отслеживает изменения DOM через MutationObserver (debounce 1000ms)
- Перехватывает SPA навигацию (pushState, replaceState, popstate)
- Показывает тултип при наведении с использованием Popover API (всегда поверх)
- Создаёт плавающую кнопку-глаз для быстрого переключения

### Popup (popup.html + popup.js)
- Компактный интерфейс (320px ширина)
- Управление URL с поддержкой wildcard
- Список атрибутов с toggle, добавлением, удалением, drag & drop сортировкой
- Выбор языка из выпадающего списка
- Автоопределение языка браузера при первом запуске

### Storage
- `allowedUrls` - массив разрешённых доменов/паттернов
- `customAttributes` - массив атрибутов (с суффиксом :disabled для выключенных)
- `language` - выбранный язык (en/ru/zh)
- `eyePosition` - позиция плавающей кнопки

## Стили подсветки

- **По умолчанию:** тонкий полупрозрачный контур (1px)
- **При наведении:** усиленный контур (2px solid red) + эффект свечения
- **Тултип:** тёмный фон с белым текстом, использует Popover API для верхнего слоя

## Добавление нового языка

Для добавления нового языка:
1. Добавьте ключ в объект `translations` в файле `locales.js`
2. Перезагрузите расширение в Chrome

Пример:
```javascript
fr: {
    name: "Français",
    title: "TestID Highlighter",
    // ... добавьте остальные поля
}
```
